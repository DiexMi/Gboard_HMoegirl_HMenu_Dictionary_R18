## 扩容与版本说明

* 保留了原有词条拼音完整性的Classic基础版本。

* 增加了前置候选的High高敏版本，即，将词段与第一个字的拼音相匹配。

* 对于某几个不易输入的词条，增加几段通俗称呼的拼音作为匹配项。

>例：惠方真姬/惠方卷，输入roubang，shousi，huifangzhenji，都可以匹配到
